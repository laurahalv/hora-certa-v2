# 🩺 Hora Certa V2

Sistema web para gerenciamento clínico e acompanhamento de pacientes, permitindo controle de médicos, configurações do sistema, notificações e registros de informações médicas.

## 📌 Sobre o projeto

O **Hora Certa V2** foi desenvolvido com o objetivo de facilitar a organização e gerenciamento de clínicas e profissionais da saúde, oferecendo uma interface intuitiva para controle de informações e acompanhamento de dados.

O sistema busca centralizar recursos como:

* Cadastro de médicos
* Gerenciamento de perfil
* Configurações da clínica
* Configurações de segurança
* Sistema de notificações
* Registro de observações e sintomas
* Interface responsiva

---

## 🚀 Funcionalidades

### 👨‍⚕️ Médicos

* Cadastro de médicos
* Informações pessoais
* Especialidade
* CPF
* Contato
* Endereço

### 🔔 Notificações

* Ativar/desativar notificações por e-mail
* Ativar/desativar notificações push
* Configurar lembretes antecipados

### 🔒 Segurança

* Alteração de senha
* Confirmação de credenciais

### 👤 Perfil

* Atualização de informações pessoais
* E-mail
* Telefone
